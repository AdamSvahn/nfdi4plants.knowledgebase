Some features you know from ARCitect are now available as a [VSCode](https://code.visualstudio.com/Download) extension. 

## VSCode extension

0. download from GitHub release page https://github.com/nfdi4plants/arc-vs-code/releases/latest
    - `arc-vs-code-x.x.x.vsix`
1. install in VSC local
  - extensions
  - three-dots menu
  - "Install from VSIX", select the dowloaded .vsix file

## VS Code extension in DataHUB "Web IDE"

0. in user prefs: activate vscode extensions 
1. open ARC in Web IDE
2. go to extensions (left sidebar) 
3. search for arc-vs-code and click "install"

## User manual

1. right click on study / assay name or isa.investigation.xlsx to delete (from file tree) or edit (Swate view)
2. Right-click on `studies` or `assays` to add a new study or assay.
