## Demo Time: Sugar Assay Data

- in "SugarMeasurement" table, open "Data Annotator" tool
    - open sugar data file
    - in the top switch from "autodetect" to "output"
    - Select only the 6(!) Data columns, as these are the direct results of the 6 samples
    - insert
- Go back to assay metadata and add Datamap... In the datamap
    - Open "Data Annotator" tool again and from the same file select all 7 columns
    - First column:
        - Explication = `molecule name`
        - Object type = `string`
    - Other columns:
        - Explication = `molecule count`
        - Unit = `millimole per kilogram`
        - Object type = `float`
