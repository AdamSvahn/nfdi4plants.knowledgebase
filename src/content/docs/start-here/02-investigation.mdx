The investigation level allows you to record project metadata such as the project title and description, responsible contacts and associated publications.

<Steps>

1. Inside **ARCitect**, click on the ARC's name on top of the file tree
2. The right panel shows the investigation form

</Steps>


<Card title="Investigation Metadata">

##### Identifier

:::note
The Investigation identifier was automatically added based on your ARC's name
:::

##### Title

- Meaningful project title (with blanks)
- Avoid administrative metadata

##### Description

- Concise project abstract 

##### Contacts

- Persons responsible for the project 

</Card>
