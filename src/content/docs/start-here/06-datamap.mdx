<Card icon="pen" title="ARCitect">
  <Steps>
  1. In the "SugarMeasurement" assay, open the "Sugar Measurement" table
  2. On top of the annotation table, open the "Data Annotator" tool
  3. Click `Choose a file`, find and select `sugar_result.csv` and click `Open`
  4. On top switch from "autodetect" to "output"
  5. Select only the 6(!) Data columns (`RT_M1`, `RT_M2`, `RT_M3`, `Cold_M1`, `Cold_M2`, `Cold_M3`), as these are the direct results of the 6 samples
  6. Click `Submit`
  7. Go back to assay metadata and add Datamap
  8. In the datamap (table sheet at the bottom)
      - Open "Data Annotator" tool again and from the same file select all 7 columns
      - First column:
          - Explication = `molecule name`
          - Object type = `string`
      - Other columns:
          - Explication = `molecule count`
          - Unit = `millimole per kilogram`
          - Object type = `float`

  </Steps>
</Card>
