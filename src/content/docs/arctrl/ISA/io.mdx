## XLSX

#### Reading

<Tabs syncKey="pl">

{/* FSHARP */}
<TabItem label="FSharp" icon='seti:'>

```fsharp
open ARCtrl

let assay = XlsxController.Assay.fromXlsxFile("myPath.xlsx")
let study,_ = XlsxController.Study.fromXlsxFile("myPath.xlsx") // the second value are registered assays
let investigation = XlsxController.Investigation.fromXlsxFile("myPath.xlsx")
let datamap = XlsxController.Datamap.fromXlsxFile("myPath.xlsx")
```
</TabItem>
