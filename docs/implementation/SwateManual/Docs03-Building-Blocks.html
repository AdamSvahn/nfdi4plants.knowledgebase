Building Blocks

last updated at 2022-09-19

Building Blocks are necessary to sustain ontology information in an annotation table and are designed to follow the core idea behind the ISA-Tab format, although simplified. For more details, please check out the Building Block Types.

Usage in Swate
  1. Navigate to the Building Blocks tab in the navbar.
  2. This tab offers two functions, which allow you to
    1. add a new building block or
    2. add units to building blocks.

DataPLANT Support

Besides these technical solutions, DataPLANT supports you with community-engaged data stewardship. For further assistance, feel free to reach out via our helpdesk or by contacting us directly .
